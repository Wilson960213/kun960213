🔍 完整操作步骤（图文级描述）
1. 进入扩展管理页面
打开 Chrome 浏览器

地址栏输入 chrome://extensions 并回车

2. 打开开发者模式（最关键）
看页面右上角，有一个 “开发者模式” 的开关

把它打开（按钮变蓝/向右）

打开后，页面左上角会出现三个按钮：
“加载已解压的扩展程序” 、 “打包扩展程序” 、 “更新”

如果你没看到这三个按钮，就是开发者模式没开。

3. 加载你的扩展文件夹
点击 “加载已解压的扩展程序”

在弹出的文件选择窗口中，找到你的项目文件夹 focuspaw-extension

注意：你要选的是整个文件夹本身，而不是点进去选中某个文件

选中文件夹后，点击“选择文件夹”（Windows）/“打开”（Mac）

4. 检查是否加载成功
加载成功后，扩展列表中会出现 FocusPaw 的卡片

如果卡片显示 🟠 错误（红色感叹号），点开“错误”看具体提示

大概率是 manifest.json 路径写错，或者缺少 background.js，或者图片路径问题

❓ 为什么你可能找不到？
最常见：开发者模式没开，按钮在页面左上角，灰色的，不显眼

第二常见：你选文件夹时，选的是 focuspaw-extension 里面的子文件夹（比如 modules），
一定要选最外层的 focuspaw-extension，里面能直接看到 manifest.json 的那个目录



操作步骤（10 秒搞定）
确保扩展已安装：在 Chrome 地址栏输入 chrome://extensions，确认 FocusPaw 已加载并开启。

点击扩展图标：浏览器右上角拼图图标 → 找到 FocusPaw 的猫猫头 → 点击它（或直接点图钉把它固定到工具栏）。

弹出小窗口：会显示一个简单的设置面板，其中有一个 密码输入框（id="apiKeyInput"）和一个 💾 保存按钮。

粘贴你的 DeepSeek API Key 到输入框里，点击保存。

页面会显示 “API Key 已保存！”

此时猫咪就能正常使用 AI 说话功能了。

你贴的代码第二段（chrome.storage.local.get）会在每次打开弹出窗口时，自动把你之前保存过的 Key 回填到输入框，方便查看是否已填。

文件夹不叫这个名字：检查你实际在电脑上建的文件夹名，必须和选择时一致
